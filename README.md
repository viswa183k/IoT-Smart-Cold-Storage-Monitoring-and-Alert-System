# IoT Smart Cold Storage Monitoring System

This project monitors temperature and humidity inside a cold storage facility using an ESP32 and DHT22 sensor. Sensor data is uploaded to ThingSpeak for remote monitoring. If the temperature goes outside the safe range, a buzzer provides an immediate local alert.

## Features

- Real-time temperature monitoring
- Humidity monitoring
- Cloud data logging using ThingSpeak
- Automatic over-temperature and under-temperature alert
- Industrial cold storage monitoring

## Hardware Required

- ESP32 Development Board
- DHT22 Temperature & Humidity Sensor
- Active Buzzer
- WiFi Connection
- Breadboard & Jumper Wires

## Applications

- Food storage
- Vaccine storage
- Pharmaceutical warehouses
- Dairy industry
- Industrial refrigeration monitoring

## Future Improvements

- SMS & Email Alerts
- Mobile Dashboard
- Predictive Cooling Failure Detection
